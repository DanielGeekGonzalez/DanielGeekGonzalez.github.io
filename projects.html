Real Estate Investment Tool

A prototype that extracts key financial data from real estate T-12 PDFs and generates core deal-screening metrics with an Excel output, reducing manual work in early-stage underwriting.

  • Tech: Python, FastAPI, Jinja2, GitHub Models, pdfplumber, openpyxl
  • Highlights:
    • Parses inconsistent, annotation-heavy T-12 PDFs into structured financial inputs
    • Computes standard metrics and exports a live Excel model
    • Uses AI only for extraction, keeping calculations deterministic and auditable

Telemetry Decoder

An application that generates, validates, and decodes binary telemetry packets using CRC-8 integrity checks. The project simulates aerospace-style data pipelines and reinforced my understanding of low-level binary data handling, system reliability, and modular software design.

  • Tech: C++17, Bash
  • Highlights:
    • Implemented CRC-validation to detect corrupt binary data before decoding
    • Designed a modular parser to safely convert raw byte streams into structured data
    • Logged decoded results to both the console and Excel for human and programmatic analysis

PyChat Server

Multi-client chat server that enables authenticated users to communicate through private messages and chat rooms, while safely handling concurrent connections and disconnects.

  • Tech: Python, socket programming, multithreading, TCP/IP
  • Highlights:
    • Implemented a threaded socket server supporting simultaneous clients with authentication, chat rooms, and direct messaging
    • Designed thread-safe shared state using locks to prevent race conditions across users and rooms
    • Built robust command parsing and error handling to gracefully manage malformed input and abrupt disconnects
